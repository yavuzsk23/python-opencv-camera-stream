# OpenCV Camera Stream

Real-time webcam stream application using OpenCV.

---

## 🇬🇧 English

**Description**  
A simple and clean Python application that captures and displays a real-time video stream from the webcam using OpenCV. The application features a custom window title and proper resource management.

**Features**
- Real-time video streaming from webcam
- Custom window title (`DE_yavuzsk08236_TR`)
- Graceful exit with 'q' key
- Automatic resource cleanup

**Requirements**
- Python 3.8+
- OpenCV (`pip install opencv-python`)

**How to Run**
python camera_stream.py

press q to exit.
