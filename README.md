# OpenCV Camera Stream Application (DE_yavuzsk08236_TR)

---

##  English

### Description
A simple and efficient Python application that utilizes the OpenCV library to capture and display a real-time video stream from the built-in webcam. The application features a custom environment identifier window and handles resources clean-up properly upon exit.

### Features
- Real-time video frame polling from webcam.
- Custom window title (`DE_yavuzsk08236_TR`).
- Graceful termination using the 'q' key.

### Requirements
- Python 3.12
- OpenCV (`opencv-python`)

## 🇩🇪 Deutsch

Eine einfache und effiziente Python-Anwendung, die die OpenCV-Bibliothek verwendet, um einen Echtzeit-Videostream von der integrierten Webcam zu erfassen und anzuzeigen. Die Anwendung verfügt über ein Fenster mit einer benutzerdefinierten Umgebungskennung und gibt die Ressourcen beim Beenden ordnungsgemäß frei.

- Benutzerdefinierter Fenstertitel (`DE_yavuzsk08236_TR`).
- Sicheres Beenden über die Taste 'q'.
